# 寧波話兩分輸入方案

配方：℞ **ionkaon/rime-gninpou-lianfen**

寧波話兩分輸入方案基於 [字海](http://zisea.com/) 兩分輸入方案及 [寧波話吳語拼音輸入方案](https://github.com/NGLI/rime-wugniu_gninpou) 製作。

## 使用

使用兩個部件的寧波話發音來輸入一個漢字。例如：

- 蒛 = 艹 (tshau) + 缺 (chiuq) = tshauchiuq
- 蒜 = 艹 (tshau) + 示 (zy) = tshauzy

以下部件的讀音需要特別注意：

- 乙：ciq（音折）
- 丶：ti（音點）
- 亻：gnin（音人）
- 攵：ven（音文）
- 阝：gni（音耳）
- 辶：tseu（音走）
- 镸：jian（音長）
- 虫：dzon（音蟲）
- 丨：zyu（音豎）
- 糸、纟、糹：sy（音絲）
- 厂：chian（音廠）
- 广：kuaon（音廣）
- 宀：ke（音蓋）
- 冖：ke（音蓋）

兩分拆分的具體細節可以查看 [兩分手冊](http://cheonhyeong.com/File/LiangFenHandbook.pdf)。
